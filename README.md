# Vyhni se!

Jednoduchá 2D webová hra vytvořená jako školní projekt v jazyce JavaScript.

## Popis projektu
Hráč ovládá modrý čtverec pomocí šipek doleva a doprava.
Z horní části obrazovky padají různé překážky, kterým se musí vyhnout.
Cílem hry je získat co nejvyšší skóre.

Projekt slouží jako ukázka práce s:
- jazykem JavaScript  
- grafickým prvkem canvas  
- objektovým programováním (class)  

## Použité technologie
- HTML  
- CSS + Bootstrap  
- JavaScript  
- Canvas  
- Git  
- GitHub Pages  

## Ovládání
- ← pohyb doleva  
- → pohyb doprava  
- Tlačítko „Restart“ – nová hra  

## Struktura projektu
- index.html – hlavní HTML stránka  
- style.css – stylování aplikace  
- script.js – logika hry  
- README.md – popis projektu
